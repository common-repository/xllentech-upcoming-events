== Description ==

Xllentech Upcoming Events Plugin shows fixed number of upcoming Islamic events at any time. It stores list of events in an array in php file. The plugin checks for the upcoming events in the file and shows fixed number of upcoming events throughout the year. 

It's very easy to add or remove event from the data file with php editor. The plugin doesn't perform database operation that makes it very efficient. 

Please Note that this plugin requires [Xllentech English Islamic Calendar plugin](https://wp-plugins.xllentech.com/xllentech-english-islamic-calendar/) installed, because of it's use for Islamic dates.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/xllentech-upcoming-events` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Make sure you have Xllentech English Islamic Calendar plugin installed, active and error free.
4. Copy xllentech-calendar-data.php file inside this plugin folder, paste it to active theme directory(Ideally child theme folder)
5. Edit the xllentech-calendar-data.php file with your events data, make sure to not distort the format of the event lines
6. Goto Widgets section in Wordpress admin login to place the widget, Give it Title as you like.
